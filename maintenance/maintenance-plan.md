# Maintenance Plan Sistem Toko Online

## Corrective
1. Memperbaiki bug login ketika pengguna memasukkan password yang benar tetapi tetap gagal masuk.
2. Memperbaiki kesalahan perhitungan total harga pada halaman checkout.
3. Memperbaiki error saat pengguna mengunggah gambar produk berukuran besar.

## Adaptive
4. Menyesuaikan sistem dengan perubahan API pembayaran online.
5. Menyesuaikan tampilan aplikasi agar kompatibel dengan browser versi terbaru.
6. Menyesuaikan sistem dengan perubahan kebijakan perlindungan data pengguna.

## Perfective
7. Menambahkan fitur pencarian produk berdasarkan kategori.
8. Meningkatkan kecepatan loading halaman katalog produk.
9. Menambahkan fitur notifikasi status pesanan kepada pelanggan.

## Preventive
10. Melakukan refactoring kode agar lebih mudah dipelihara.
11. Memperbarui dokumentasi penggunaan sistem.
12. Membuat backup database secara otomatis setiap hari.
