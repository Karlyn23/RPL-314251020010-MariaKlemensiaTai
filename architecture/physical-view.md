# Physical View

## Deskripsi

Physical View menggambarkan perangkat keras dan perangkat lunak yang digunakan untuk menjalankan Sistem Toko Online.

## Perangkat Pengguna

- Laptop
- Komputer
- Smartphone

Mengakses sistem menggunakan:
- Google Chrome
- Microsoft Edge
- Mozilla Firefox

## Web Server

Server menjalankan aplikasi Sistem Toko Online.

Teknologi:
- HTML
- CSS
- JavaScript
- PHP

## Database Server

Database menggunakan MySQL.

Data yang disimpan:
- User
- Produk
- Kategori
- Pesanan
- Pembayaran

## Jaringan

Perangkat pengguna
↓

Internet
↓

Web Server
↓

Database Server
