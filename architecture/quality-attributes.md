# Quality Attributes

## 1. Performance
Sistem harus dapat menampilkan halaman produk dengan cepat agar pengguna tidak menunggu terlalu lama.

## 2. Security
Sistem harus melindungi data pengguna seperti password, alamat, dan informasi pembayaran.

## 3. Usability
Sistem harus mudah digunakan oleh pelanggan maupun admin.

## 4. Scalability
Sistem harus mampu menangani peningkatan jumlah pengguna dan produk.

## 5. Availability
Sistem harus dapat diakses kapan saja dengan gangguan seminimal mungkin.
